# tischtennis-arduino
Ich habe einen Arduino Nano 33 BLE Sense Rev 2. Diesen habe ich in einen Tischtennisschläger eingebaut.

## Sketches
- data-collection: Zum Sammeln von Trainingsdaten in Kombination mit web app data-collection.html
- smart-table-tennis-main: Klassifizierung der Schläge; ermittelt Vorhand, Rückhand, Konter, Topspin, ...
- tt_performance: Sammelt die Daten zu Beschleunigung, Drehung und Magnetometer; in Kombination mit web app tt-web
